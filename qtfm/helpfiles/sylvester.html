Quaternion Function Reference

sylvester

Solve Sylvester's equation ax + xb = c
(Quaternion overloading of standard MATLAB® function)

Syntax

x = sylvester(a, b, c)

Description

This function solves for x. The first two parameters must be of the same size. The function works for real and complex quaternions.

It is possible for there to be no solution (singular case). If so, the return result will be NaNs.
