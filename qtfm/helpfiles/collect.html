Quaternion Function Reference

collect

Collect coefficients.
(Quaternion and octonion overloadings of standard MATLAB® function)

Syntax

C = collect(P, expr)

Description

This function requires the MATLAB® Symbolic Math Toolbox.

collect collects coefficients of each component of its quaternion or octonion parameter expr by application of the MATLAB® function of the same name to each component of the quaternion or octonion P. It accepts the same parameter profiles as the overloaded MATLAB® function.
