Quaternion Function Reference

assume

Set assumptions on symbolic quaternion or octonion.
(Quaternion and octonion overloadings of standard MATLAB® function)

Syntax

assume(expr, varargin)

Description

This function requires the MATLAB® Symbolic Math Toolbox.

assume sets an assumption on each component of its quaternion or octonion parameter expr.
