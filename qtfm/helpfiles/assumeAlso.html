Quaternion Function Reference

assumeAlso

Add assumptions on symbolic quaternion or octonion.
(Quaternion and octonion overloadings of standard MATLAB® function)

Syntax

assumeAlso(expr, varargin)

Description

This function requires the MATLAB® Symbolic Math Toolbox.

assumeAlso adds an assumption on each component of its quaternion or octonion parameter expr.
