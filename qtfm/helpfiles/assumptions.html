Quaternion Function Reference

assumptions

Show assumptions on components of symbolic quaternion or octonion.
(Quaternion and octonion overloadings of standard MATLAB® function)

Syntax

assumptions(q)

Description

This function requires the MATLAB® Symbolic Math Toolbox.

assumptions lists assumptions that apply to each component of the quaternion or octonion q. See the documentation on the MATLAB® function of the same name to see how it is used.
