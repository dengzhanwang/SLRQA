Quaternion Function Reference

expand

Expand expressions
(Quaternion and octonion overloadings of standard MATLAB® function)

Syntax

Y = expand(S)

Description

This function requires the MATLAB® Symbolic Math Toolbox.

expand works by application of the MATLAB® function of the same name to each component of the quaternion or octonion S. It accepts the same parameter profiles as the overloaded MATLAB® function.
